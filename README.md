# AI Python Exercises

This repository contains AI related exercises and small problem solutions developed as part of my university course.

## Exercise Descriptions

**Exercise01**
- Dataset: diabetes.csv  
- Task: Predict diabetes outcome with KNN algorithm.
- Techniques: Data visualization, normalization, KNN

**Exercise02**
- Dataset: multilinearregression.csv
- Task: Predict house prices based on area, room count, and building age using Multilinear Regression.
- Techniques: Data preprocessing, Multilinear Regression modeling, coefficient analysis

**Exercise03**
- Dataset: polynomial.csv 
- Task: Predict salary based on years of experience.
- Techniques: Data visualization, Linear Regression vs. Polynomial Regression, Polynomial Feature transformation.

**Exercise04**
- Dataset: pca_iris.data 
- Task: Visualize high-dimensional data in 2D.
- Techniques: Standardization, Principal Component Analysis, Feature projection, Scatter plot visualization.

**Exercise05**
- Dataset: Avm_Musterileri.csv 
- Task: Customer segmentation based on annual income and spending score.
- Techniques: Data normalization, Elbow Method to determine optimal cluster number, K-Means algorithm, cluster centroid visualization.

**Exercise06**
- Dataset: users.data & movie_id_titles.csv  
- Task: Build a simple movie recommendation engine based on user rating correlations.
- Techniques: Data merging, Pivot Tables, Correlation analysis, Data filtering by rating counts.
